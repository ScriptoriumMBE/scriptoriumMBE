# Tres danzas chilenas (2023)
Pieza para dos violines. Escrita en 2023.

Duración total: ca. 3 minutos

A la fecha, la pieza es inédita.

Para conseguir las partes y/o interpretar la pieza, por favor escriba a
be-mikael@proton.me

## Nota de programa
Esta pieza contiene tres pequeños momentos, cada uno representando la música de una zona de Chile. Para el sur, elegí la música del pueblo Mapuche. Para el centro, elegí la danza nacional, la cueca. Para el norte elegí la música andina.
